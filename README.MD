此书完全是三位师兄因共同爱好而编写，如果时间充裕，最好直接看成观法师与圆瑛法师的原著。
编写此书的因缘是因为现在人生活节奏快，时间紧张，碎片化信息多的缘故而编写此书，此书参考成观法师与圆瑛法师的楞严经著作而提炼精简，作者再重述为解释部分并校对发布，
内容上确保忠于两位法师的本意， 编写此书的目的有两个：短，能懂。节约时间的同时也能愉快的学习。
